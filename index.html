Assignment 2: Trends on Global Temperature

 

1. Data Exploration

            First, several libraries are installed and utilized for this project: ‘nycflight13’, ‘ggplot2’, ‘tidyr’, and ‘dplyr’. These libraries include the source data, tools for cleaning as well as visualizing. Then, we proceed to the exploration of the data to figure out what steps should be taken such as which variables can be omitted to reduce noise, the length of the data, the number of missing values, the summary of its statistics, etc.

            There are a total of 15 columns with 26,115 observations in this dataset. By looking at the variables in the data and connecting them with our topic, which is trends of global temperature, we decided to take these eight main variables: origin, time_hour, month, temp, dewp, humid, wind_speed, and precip. The reason year is omitted is because it is the same for all observations (all data being recorded in 2013).

            From this, we can also know that there are three different elements in origin which can be used later in visualization as hue or classification. The rest of the information can be added but not necessarily significant to the report on temperature.

            Lastly, we extract the columns into a new dataset named temp_data to use from now on. It consists of 8 columns and 26,115 observations.

2. Data Cleaning

            Before analyzing and visualizing further, the data was checked for any missing values, in which it did have one missing value in the temp, dewp, humid as well as several ones in wind_speed. Prior to this, we checked the summary of temp_data on the console and discovered that the mean and median of the columns were not that different, so assuming the data is not skewed, we replaced it with the mean of the values (omitting the null values).

 

            To be sure the data possess no outliers, we used the interquartile range method to gather any possible outliers. According to the IQR, there was none.

Figure 1 Boxplot of temp column from temp_data

3. Data Analysis

            We started analyzing by reviewing again the summary of the dataset. We could see that the average (mean) temperature for the whole year of 2013 was 55.26°F, with the lowest recorded at 10.94°F and highest at 100.04°F. Because there are three origins for the temperature, we wanted to view the statistical summary of these different origins as well. Since all three origins are airports from the same city, we can see that the temperature doesn’t differ much.

            To visualize the average temperature, we use the ggplot2 library to visualize a line chart with time_hour as the x axis and temp as the y axis. From the simple visualization, we could see the trend generally moves upwards during the first half of the year, and downwards during the second half.

Figure 2 Time Series of Average Temperature in NYC (2013)

4. Trend Analysis

.           As mentioned before, we assume that several variables such as dew point, humidity, wind speed and precipitation play a role in temperature, we are going to visualize it using a correlation plot. By using ggcorrplot, we can easily recognize that dewp has quite a strong correlation to temp (0.89), and month has a weaker positive correlation to temp (0.27. We could also see that wind_speed is the only negative correlation to temp, although it is considered weak (-0.11).
